# Thinking Sessions

This directory stores deep thinking session outputs from `/max-think`.

## Purpose

- Record 6-dimensional structured analysis
- Preserve strategic decision rationale
- Enable future reference and audit

## Naming Convention

- `{timestamp}-{topic}.md` - Thinking session output

## 6D Analysis Framework

1. **Technical Dimension** - Architecture, performance, scalability
2. **Business Dimension** - ROI, market fit, competitive advantage
3. **Team Dimension** - Skills, capacity, learning curve
4. **Ecosystem Dimension** - Community, support, longevity
5. **Strategic Dimension** - Long-term alignment, future-proofing
6. **Meta Dimension** - Decision confidence, reversibility
